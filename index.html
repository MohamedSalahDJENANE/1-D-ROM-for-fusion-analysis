Function Syntax Help

Available variables depend on the boundary context:

  • Material Props: t (time), T (local temp), z, x or y
  • Inlet/Outlet BCs (z=0, L): t, x or y (transverse coordinate), T
  • Top/Bottom Walls: t, z (longitudinal coordinate), T

Note: Math functions like sin, cos, exp, pow, PI can be used directly.

1000 * pow(x, 2) (or simply 1000 * x * x)
1000 * sin(PI * x / 0.05)
300 + 50 * exp(-10 * z)
if(t < 100; 350; 300)
